¿Qué es este proyecto?

Una propuesta educativa para aprender a programar, controlar hardware y crear apps usando el ESP32, servomotores y Telegram o App WiFi.

¿Qué incluye?

  • Tutoriales paso a paso en YouTube para la aplicacion y el bot de telegram
  • Códigos explicados arduino ide para ESP32
  • Guía con explicaciones
  • Lista de materiales, kit sugerido

¿Qué aprenderás?

  • Programación en Arduino Ide (C++)
  • Control de hardware: servos y pantallas
  • Conexión WiFi y uso de bots de Telegram
  • Diseño de interfaces con Mit App Inventor

Para quién es

Ideal para docentes de tecnología, talleres de robótica, estudiantes curiosos, y clubes escolares.
